# To-Do-App-in-React

Welcome to your Todo List App, a simple yet powerful tool to manage your daily tasks efficiently. Built with React, this app leverages Material-UI for modern and responsive UI components, including edit and delete icons for intuitive task management.

Key Features:

Add Todo: Users can effortlessly add new tasks to their list using the input field and the add button with a user-friendly AddIcon.

Edit Todo: Each task can be edited using the EditIcon. A modal powered by Material-UI allows users to update the task description without any page reloads, ensuring a smooth user experience.

Delete Todo: Users can easily delete tasks using the ClearIcon, keeping their todo list up-to-date with minimal effort.

State Management: The app efficiently manages state using React's useState and useEffect hooks, ensuring real-time updates and seamless user interactions.

API Integration: The app communicates with a backend server to fetch, add, update, and delete tasks using RESTful API endpoints, ensuring persistent and synchronized task management.

Components:

InputTodo: Handles the addition of new tasks with an input field and a submit button.

ListTodos: Displays the list of tasks with options to edit and delete each task.

EditTodo: Manages the editing of tasks through a modal interface, allowing users to update task descriptions.

This app demonstrates modern web development practices, including component-based architecture, state management, and asynchronous operations with React.
